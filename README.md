##  키읔피읖 카페가자 

---
원하는 카페를 가야하는데 일일이 검색해서 찾기 불편한걸 해소하고자 제작한 사이트입니다. 

최근 SNS에서 인기를 끌고있는 카페들이 많아짐으로 원하는 카테고리를 선택하여 찾을수 있다 .<br/>현재 지도를 통해 원하는 카페와 내위치간의 거리를 확인할수 있다. <br/> 20대부터 50대까지 연령별 인기있는 카페를 찾을수 있다!<br/>
